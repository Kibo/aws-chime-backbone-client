AWS Chime webclient in Backbone.js wrapper.

## Version
- 0.0.0 ( coding )

### Dependencies
- amazon-chime-sdk-js (2.5.0)
- Backbone.js (1.4.0)
- Underscore.js ( >= 1.8.3)
- jQuery ( >= 1.11.0)

### Tools
- [NPM](https://npmjs.org) - Node.js package manager.
- [Grunt](http://gruntjs.com/) - The JavaScript Task Runner.

### Before Build
- pls, see [build.config.js](#).
